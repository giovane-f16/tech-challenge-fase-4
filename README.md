# Tech Challenge Fase 4

Aplicativo de Blogging desenvolvido com Expo e React Native.

## Integração
A aplicação utiliza o projeto [Tech Challenge Fase 2](https://github.com/giovane-f16/tech-challenge-fase-2) como backend para gerenciamento de posts.

## Requisitos
1. Node.js

> https://nodejs.org/pt

2. Instancia do MongoDB Atlas

> https://www.mongodb.com/products/platform/atlas-database

3. JWT Token

> https://www.jwt.io/

4. Projeto no Firebase

> https://console.firebase.google.com/

## Como usar
1. Clone este repositório

   ```bash
   git clone https://github.com/giovane-f16/tech-challenge-fase-4.git
    ```

2. Crie o arquivo config.ts com base no [config.example.ts](./app/src/Config/config-example.ts)

3. Instale as dependências

   ```bash
   npm install
   ```

4. Inicie o aplicativo

   ```bash
   npx expo start
   ```

5. Escaneie o QR code com o aplicativo **Expo Go** no seu dispositivo móvel ou use um **emulador**.
